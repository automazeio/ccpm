# Task: Code Review Agent

## Description
Create a markdown-based agent for Claude to review Ollama-generated code. This agent will analyze code for bugs, style violations, security issues, and missed requirements, providing structured feedback to guide the review iteration process.

## Acceptance Criteria
- [ ] Markdown agent follows CCPM pattern
- [ ] Reviews code for: bugs, style violations, security issues, missed requirements
- [ ] Provides specific, actionable feedback
- [ ] Returns structured review result (approve/iterate/fail)
- [ ] Includes examples of good/bad code patterns

## Technical Details

### Implementation Approach
- Create markdown agent at `ccpm/agents/claude-code-reviewer.md`
- Define structured review format with clear sections:
  - Code quality assessment
  - Security analysis
  - Requirements validation
  - Style compliance
- Include decision matrix for approve/iterate/fail outcomes
- Provide examples of common issues and how to address them

### Key Considerations
- Agent should be strict but constructive
- Feedback must be actionable (not just "this is bad")
- Review criteria should be consistent across iterations
- Support for multiple languages/frameworks
- Balance between thoroughness and speed
